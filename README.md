# AWS CodePipeline CI/CD Showcase

A modern web application demonstrating AWS CodePipeline CI/CD workflow using GitHub, AWS CodeBuild, and AWS CodeDeploy. Built with React, TypeScript, Vite, and Tailwind CSS.

## Features

- Interactive showcase of AWS CI/CD pipeline components
- Modern UI with Framer Motion animations
- Responsive design with Tailwind CSS
- shadcn/ui components
- TypeScript for type safety

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- shadcn/ui
- Lucide React icons

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```
